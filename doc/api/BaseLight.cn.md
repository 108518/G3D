# BaseLight

光源基类。

## 属性

| 名称      | 类型                              | 描述                                      |
| --------- | --------------------------------- | ----------------------------------------- |
| color     | {r: Number, g: Number, b: Number} | 光源颜色，默认为 {r: 255, g: 255, b: 255} |
| intensity | Number                            | 光源强度，默认为 1                        |
