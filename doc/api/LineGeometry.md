# Geometry

The LineGeometry.

## Constructor

```javascript
new G3D.LineGeometry(mesh);
```

### options

| name | type     | description         |
| ---- | -------- | ------------------- |
| mesh | G3D.Mesh | the geometry's mesh |

## Properties

| name     | type              | description                 |
| -------- | ----------------- | --------------------------- |
| vertices | [Number]          | the vertices array          |
| indices  | {[key]: [Number]} | the indice array collection |
