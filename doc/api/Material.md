# Material

Base class of materials.

## Constructor

```javascript
new G3D.Material();
```

## Properties

| name    | type                   | description                                 |
| ------- | ---------------------- | ------------------------------------------- |
| opacity | Number                 | material opacity, from 0 to 1, default is 1 |
| offset  | {x: Number, y: Number} | material texture uv offset                  |
